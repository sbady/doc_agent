Категории чувствительных данных для фильтрации
1. Учетные данные и ключи доступа
text
- API-ключи (форматы: /[a-zA-Z0-9]{32,64}/)
- Токены авторизации (JWT, OAuth, Bearer tokens)
- Пароли (любые строки после "password:", "pwd:", "пароль:")
- Ключи шифрования (PEM, private keys)
- Ключи сервисов (AWS, Google Cloud, Azure keys)
- SSH-ключи
2. Персональные данные (PII)
text
- Email адреса
- Номера телефонов (российские и международные форматы)
- Паспортные данные (серии, номера)
- ФИО полностью
- Должности с привязкой к конкретным людям
3. Внутренняя инфраструктура
text
- Внутренние IP-адреса (10.x.x.x, 192.168.x.x, 172.16.x.x-172.31.x.x)
- Доменные имена внутренних систем (*.local, *.internal)
- Имена баз данных
- Пароли от баз данных и подключения
4. Финансовая информация
text
- Номера банковских карт
- Реквизиты компаний (расчетные счета)
- Суммы платежей с привязкой к конкретным сделкам
5. Конфигурации и настройки
text
- Файлы .env, config files с чувствительными данными
- Connection strings (базы данных, брокеры сообщений)
- Настройки firewall/security групп
