Глоссарий терминов платформы Mashroom
1. Платформа (Mashroom)
Продукт для проведения мероприятий. Функционал: регистрация участников, видеотрансляции, опросы, сбор статистики.

2. Административный интерфейс
Рабочее пространство организатора для управления мероприятиями. Включает: создание и настройку мероприятий, управление участниками, настройку интерактивных блоков, доступ к статистике, запуск трансляций через Студию.

3. Мероприятие
Основная техническая единица на платформе, эквивалентная виртуальному залу. Имеет собственную регистрацию, видеотрансляцию, чат и статистику.
Ключевой контекст: Реальное событие (например, конференция с параллельными секциями) состоит на платформе из нескольких мероприятий (по одному на зал). Мероприятия могут быть объединены для упрощенной регистрации.
Форматы: онлайн, офлайн, гибрид.
Типы входа: свободный, по списку, с регистрацией.

4. Участник
Пользователь, зарегистрировавшийся на мероприятие.

5. Организатор
Пользователь, создающий и проводящий мероприятия.

6. Администратор
Пользователь с расширенными правами для управления организаторами и их мероприятиями.

7. Модератор
Пользователь, назначаемый организатором для управления чатом, вопросами, опросами и расписанием во время мероприятия. На одном мероприятии — один модератор.

8. Код доступа
Персональный код участника для авторизации и доступа к мероприятию.

9. Лимит активаций кода
Максимальное количество использований персонального кода доступа участника с разных устройств.

10. Секретный код
Многоразовый универсальный код доступа к мероприятию без ограничений по активациям.

11. Постер
Статическое или анимированное изображение, отображаемое в плеере до начала трансляции.

12. Лендинг (landing)
Одностраничный сайт мероприятия. По умолчанию генерируется платформой (MID), но может быть встроен на сайт организатора.

13. Виджет
Интерактивный элемент платформы для встраивания на сторонние сайты. Типы: платформы, регистрации, заказов.

14. Виджет платформы
Виджет, содержащий плеер трансляции и интерактивный блок.

15. Виджет регистрации
Виджет для встраивания формы регистрации на сторонний сайт.

16. Виджет заказов
Виджет для встраивания формы продажи билетов на платные мероприятия.

17. Интерактивный блок
Часть виджета платформы для взаимодействия с участниками. Элементы: чат, опросы, вопросы, расписание, список участников и др.

18. Встройка (embed)
HTML-код для интеграции виджетов платформы (регистрации, плеера) на сторонний сайт.

19. Ключ потока
Уникальный идентификатор, связывающий источник видеосигнала (внешний encoder) с конкретным мероприятием на платформе.

20. Студия
Встроенный в платформу инструмент для проведения трансляций прямо из браузера (альтернатива OBS+Zoom).
