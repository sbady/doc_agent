# Релиз T1 Disk и Dion Disk 1.27

Дата релиза: 14.05.2026

В релиз вошли улучшения T1 Disk и Dion Disk: исправления в работе публичных ссылок, предпросмотра файлов, поиска, операций с папками и файлами, а также обновления административных сценариев управления пользователями и компаниями.

## T1 Disk

### Работа с файлами и папками

- При переименовании файла расширение больше не попадает в редактируемую часть названия. Это снижает риск случайно изменить формат файла
- В пустой папке в режиме "Плитка" теперь отображается корректная заглушка и доступно действие добавления файла
- После создания папки окно создания закрывается автоматически, без необходимости закрывать его вручную
- При создании папки с уже существующим названием во время перемещения теперь отображается понятная ошибка
- Исправлено скачивание изображения из окна предпросмотра при открытии файла по публичной ссылке

### Управление пользователями и квотами

- Исправлено отображение пользовательской квоты: остаток больше не показывается как отрицательное значение
- При разблокировке пользователя корректно учитывается доступная квота компании
- Пользователи в статусе "Готов к удалению" больше не занимают пользовательскую квоту
- Исправлено отображение статуса пользователя, готового к удалению: вместо двойного статуса показывается один корректный статус
- Исправлен сценарий удаления пользователя: окно удаления открывается корректно, а пользователь не переходит в статус, который блокирует ручное удаление
- Подсказка о сроке окончательного удаления пользователя теперь показывает значение, настроенное для конкретной конфигурации

### Администрирование компаний

- В административном интерфейсе T1 Disk доработано создание и редактирование компании: можно указать квоту, максимальное количество пользователей и связать компанию с организацией Dion

## Dion Disk

### Публичные ссылки

- Исправлено скачивание файлов по публичной ссылке через контекстное меню
- PDF-файлы по публичным ссылкам теперь открываются в предпросмотре, в том числе для ссылок с ограничением "Только предпросмотр" и ссылок с паролем
- По публичной ссылке с паролем теперь корректно открываются вложенные папки
- Для публичных ссылок с истекшим сроком действия или исчерпанным лимитом скачиваний теперь отображается понятная ошибка вместо бесконечной загрузки или нерабочей формы ввода пароля
- Для публичных ссылок с паролем исправлено отображение формы ввода пароля и ошибки при неверном пароле
- По публичной ссылке с правом загрузки теперь работает drag-and-drop загрузка файлов
- В разделе "Публичные файлы" добавлена возможность скачать файл или папку через контекстное меню
- В разделе "Публичные файлы" при отсутствии файлов теперь отображается корректная подсказка

### Доступы и совместная работа

- Для папок, доступных только на чтение, больше не отображаются действия редактирования, загрузки и создания папок
- Аналогичное исправление внесено для папок компании с правами только на чтение
- При переносе папки поиск нового владельца теперь корректно показывает найденного пользователя
- В окне переноса папки исправлен формат отображения выбранного нового владельца
- Исправлен текст события при приглашении пользователя в общую папку
- Счетчик новых событий теперь обновляется до перехода пользователя в раздел "События"

### Файлы, папки и поиск

- Drag-and-drop загрузка нескольких файлов и папок теперь обрабатывает все выбранные элементы, а не только один
- Для нескольких выбранных папок снова доступно действие переноса
- После последовательного создания нескольких папок список обновляется без перезагрузки страницы
- Предпросмотр файла теперь открывается сразу после загрузки, без обновления страницы
- Поиск по тексту теперь находит загруженные файлы, даже если они не редактировались через диск
- При переименовании файла или папки в уже существующее название теперь отображается понятный текст ошибки
- Список запрещенных символов для названий файлов и папок дополнен и применяется при вводе названия

### Версии файлов и предпросмотр

- Исправлено воспроизведение видео после восстановления версии файла
- Видеоплеер в окне версий больше не перекрывается панелью версий
- Действие "Показать версии" корректно работает из контекстного меню

### Папки компании и доступ к Dion Disk

- После запроса доступа к папке компании статус "Запрошено" отображается без обновления страницы
- Доступ к Dion Disk теперь учитывает корпоративные политики Dion: при включенной политике доступ доступен, при выключенной — недоступен
