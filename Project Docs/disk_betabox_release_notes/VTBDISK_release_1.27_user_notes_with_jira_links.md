# Релиз T1 Disk и Dion Disk 1.27

Дата релиза: 14.05.2026

В релиз вошли улучшения T1 Disk и Dion Disk: исправления в работе публичных ссылок, предпросмотра файлов, поиска, операций с папками и файлами, а также обновления административных сценариев управления пользователями и компаниями.

## T1 Disk

### Работа с файлами и папками

- [VTBDISK-1103](https://jira.t1-cloud.ru/browse/VTBDISK-1103) - При переименовании файла расширение больше не попадает в редактируемую часть названия. Это снижает риск случайно изменить формат файла
- [VTBDISK-1863](https://jira.t1-cloud.ru/browse/VTBDISK-1863) - В пустой папке в режиме "Плитка" теперь отображается корректная заглушка и доступно действие добавления файла
- [VTBDISK-2461](https://jira.t1-cloud.ru/browse/VTBDISK-2461) - После создания папки окно создания закрывается автоматически, без необходимости закрывать его вручную
- [VTBDISK-2437](https://jira.t1-cloud.ru/browse/VTBDISK-2437) - При создании папки с уже существующим названием во время перемещения теперь отображается понятная ошибка
- [VTBDISK-2458](https://jira.t1-cloud.ru/browse/VTBDISK-2458) - Исправлено скачивание изображения из окна предпросмотра при открытии файла по публичной ссылке

### Управление пользователями и квотами

- [VTBDISK-2301](https://jira.t1-cloud.ru/browse/VTBDISK-2301) - Исправлено отображение пользовательской квоты: остаток больше не показывается как отрицательное значение
- [VTBDISK-2459](https://jira.t1-cloud.ru/browse/VTBDISK-2459) - При разблокировке пользователя корректно учитывается доступная квота компании
- [VTBDISK-2466](https://jira.t1-cloud.ru/browse/VTBDISK-2466) - Пользователи в статусе "Готов к удалению" больше не занимают пользовательскую квоту
- [VTBDISK-2471](https://jira.t1-cloud.ru/browse/VTBDISK-2471) - Исправлено отображение статуса пользователя, готового к удалению: вместо двойного статуса показывается один корректный статус
- [VTBDISK-2469](https://jira.t1-cloud.ru/browse/VTBDISK-2469) - Исправлен сценарий удаления пользователя: окно удаления открывается корректно
- [VTBDISK-2468](https://jira.t1-cloud.ru/browse/VTBDISK-2468) - Пользователь, готовый к удалению, больше не переходит в статус, который блокирует ручное удаление
- [VTBDISK-2402](https://jira.t1-cloud.ru/browse/VTBDISK-2402) - Подсказка о сроке окончательного удаления пользователя теперь показывает значение, настроенное для конкретной конфигурации

### Администрирование компаний

- [VTBDISK-2435](https://jira.t1-cloud.ru/browse/VTBDISK-2435) - В административном интерфейсе T1 Disk доработано создание и редактирование компании: можно указать квоту, максимальное количество пользователей и связать компанию с организацией Dion

## Dion Disk

- [VTBDISK-2199](https://jira.t1-cloud.ru/browse/VTBDISK-2199) - По публичной ссылке с правом загрузки теперь работает drag-and-drop загрузка файлов
- [VTBDISK-2187](https://jira.t1-cloud.ru/browse/VTBDISK-2187) - В разделе "Публичные файлы" добавлена возможность скачать файл или папку через контекстное меню
- [VTBDISK-1956](https://jira.t1-cloud.ru/browse/VTBDISK-1956) - В разделе "Публичные файлы" при отсутствии файлов теперь отображается корректная подсказка

- [VTBDISK-1635](https://jira.t1-cloud.ru/browse/VTBDISK-1635) - В окне переноса папки исправлен формат отображения выбранного нового владельца
- [VTBDISK-2203](https://jira.t1-cloud.ru/browse/VTBDISK-2203) - Исправлен текст события при приглашении пользователя в общую папку
- [VTBDISK-2204](https://jira.t1-cloud.ru/browse/VTBDISK-2204) - Счетчик новых событий теперь обновляется до перехода пользователя в раздел "События"
